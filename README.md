# Gesture Orchestra

Gesture Orchestra is an interactive web-based sound system that allows users to generate and control audio using hand gestures.

The application maps user interactions (mouse or touch input) to different sound outputs, creating a dynamic and engaging audio experience directly in the browser.

---

## Overview

This project demonstrates how user gestures can be translated into sound using browser-based technologies. It focuses on real-time interaction, responsive audio feedback, and intuitive control mechanisms.

---

## Features

- Gesture-based sound triggering
- Real-time audio feedback
- Interactive browser experience
- Lightweight and responsive design

---

## Tech Stack

- HTML
- CSS
- JavaScript (Web Audio / DOM interaction)

---

## Project Structure

gesture-orchestra/

## How to Use

1. Open `index.html` in a browser  
2. Interact using mouse or touch  
3. Explore different gesture-based sound responses  

---

## Notes

- Ensure all files remain in the same structure for sounds to load correctly
